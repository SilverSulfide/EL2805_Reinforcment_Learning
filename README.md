# EL2805_Reinforcment_Learning
EL2805 at KTH lab solutions by markuz@kth.se and amper@kth.se

## Lab 1: MDP's

### Problem 1:
Escape the maze without being eaten by the Minotaur. All required classes and functions are contained within
**maze.py**.

#### Basic call
```
python main.py --problem maze
```
#### Optional arguments
```
--stay Allow 'stay' action for the minotaur
--jump Force minotaur to jump over walls instead of walking through them
```
### Problem 2:
Rob banks in a city without being caught by the police. All required classes are contained within
**city.py**.

#### Basic call
```
python main.py --problem city
```
